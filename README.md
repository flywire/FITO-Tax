# Australian FITO Tax Calculator

This is a small, static HTML tax calculator for Australian residents with foreign‑sourced income, designed to illustrate how the **Foreign Income Tax Offset (FITO)** interacts with income tax and Medicare Levy.

Live demo:  
👉 [https://flywire.github.io/FITO-Tax](https://flywire.github.io/FITO-Tax)

## Features

- Calculates gross tax, Low‑Income Tax Offset (LITO), Foreign Income Tax Offset (FITO), income tax, Medicare Levy, and total amount payable.
- Demonstrates how foreign‑sourced income tax affects FITO and how FITO reduces income tax and Medicare Levy.
- The “Check Constants” button lets you inspect the underlying tax bands, rates, thresholds, and FITO‑limit values.
- Runs entirely in the browser as a static page; no backend or API required.
