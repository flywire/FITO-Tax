# Australian FITO Tax Calculator

Tax calculator for Australian residents with foreign‑sourced income, designed to demonstrate **Foreign Income Tax Offset (FITO)** interaction with income tax and Medicare Levy.

**Live:** 👉 https://flywire.github.io/FITO-Tax

Alternatively,

**Demo link:** 👉 https://flywire.github.io/FITO-Tax?taxable=50000&foreignInc=15000&foreignTax=2000

## Features

- Calculates gross tax, Low‑Income Tax Offset (LITO), Foreign Income Tax Offset (FITO), income tax, Medicare Levy, and total amount payable.
- Demonstrates how foreign‑sourced income tax affects FITO and how FITO reduces income tax and Medicare Levy.
- "🔗 Link" button generates shareable URLs with your input values.
- Use “Check Constants” button to ensure current underlying tax bands, rates, thresholds, and FITO‑limit values.
- Runs entirely in the browser as a static page; no backend or API used.
